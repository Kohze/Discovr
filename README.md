# Discovr
Graphical clusters of a variety of statistical analysis methods

Statistical Test Included 
--------------

- Welchs T.Test
- Shapiro Wilks Test
- Correlation Test
- Anova Test
- Chi Square Test
- Wilcoxon Test
- Mann-Whitney U test



Discovr Shiny App
--------------

The app serves as interactive tool to learn and understand
the discovr syntax.
